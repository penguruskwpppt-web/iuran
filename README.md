# Aplikasi Iuran Warga

Sistem manajemen iuran warga berbasis web dengan backend Supabase.

## Fitur
- Manajemen data warga (CRUD)
- Pencatatan iuran bulanan dengan upload bukti transfer
- Laporan kas & riwayat transaksi
- Dashboard statistik
- Admin panel untuk manajemen user dan backup data

## Teknologi
- HTML, CSS, JavaScript (Vanilla)
- Supabase (PostgreSQL, Auth, Storage)
- GitHub Pages (hosting)

## Cara Menjalankan Lokal
1. Clone repository ini
2. Ganti `supabaseUrl` dan `supabaseKey` di `js/app.js` dengan kredensial Supabase Anda
3. Buka `index.html` dengan Live Server atau server lokal

## Struktur Database
Lihat file `docs/database_schema.sql` untuk detail tabel.

## Deployment
Repository ini siap di‑deploy ke GitHub Pages. Cukup push ke branch `main`, lalu aktifkan GitHub Pages di Settings.
